## 공개 저장소 방향 · Philosophy

공개 저장소는 실무 프로젝트의 원본을 올리는 공간이 아니라, **개인 허브 · 독립 서비스 · 템플릿 · 실험**을 통해 *문제 정의 → 검증 → 개선* 과정을 제품 단위로 정리하는 공간입니다.

Open repos aren't mirrors of work projects — they're where I structure the *define → validate → iterate* loop as independent products, hubs, templates, and experiments.
